# twitoff-15

## Installation

Download the repo and navigate there from the command line:

```sh
git clone git@github.com:s2t2/twitoff-15.git
cd twitoff-15
```

## Setup

Setup and activate a virtual environment:

```sh
pipenv install
pipenv shell
```


## Usage

Run the web app:

```sh
FLASK_APP=web_app flask run
```
